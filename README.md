# 📝 Simple To-do List Web App

A modern, responsive **Todo List application** built using **HTML, CSS, and Vanilla JavaScript**.  
It helps users manage daily tasks efficiently with features like task filtering, local storage persistence, and a beautiful **dark/light theme toggle**.

---

## ✨ Features

- ➕ Add new tasks  
- ✏️ Edit existing tasks  
- 🗑️ Delete tasks  
- ✅ Mark tasks as completed  
- 🔍 Filter tasks (All / Pending / Completed)  
- 🌙 Dark & ☀️ Light theme toggle  
- 💾 Persistent storage using **LocalStorage**  
- 📱 Fully responsive design  
- 🎨 Custom styled checkboxes & premium UI

## 🧠 How It Works

- Tasks are stored in **LocalStorage**, so they persist after refresh  
- Theme preference is saved automatically  
- DOM manipulation is used for real-time updates  
- Filters dynamically update task views
